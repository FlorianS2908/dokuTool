# Lokale Referenzbibliothek

Dieses Verzeichnis enthaelt nur commitbare Metadaten fuer private lokale Fachbuch-Referenzen.

Die Buchdateien selbst liegen ausschliesslich lokal unter:

```text
.data/references/books/
```

Nicht committen:

- PDF-/EPUB-/E-Book-Dateien
- extrahierte Volltexte
- lange Auszuege oder Zitate aus Buechern

Commitbar:

- `reference-manifest.json`
- `reference-topics.json`
- Import-, Scan- und Validierungsskripte
- Referenz-Metadaten und Regelzuordnungen

Die Referenzen dienen als private lokale Unterstuetzung fuer Ruleset-Reviews. Sie ersetzen nicht das Ruleset und werden nicht ueber die API als Buchinhalt ausgegeben.
